Medsol
Medsol is a medical appointment booking system built using the MERN stack (MongoDB, Express.js, React.js, Node.js).
It allows patients to book, view, and manage their medical appointments easily while providing doctors with an interface to manage their schedules.

Features:- 

User authentication and authorization

Patient dashboard for booking and viewing appointments

Booking Appointment (used stripe for payment)

user-friendly UI
